# VisualKernel
VisualKernel integrates Linux kernel debugging into Visual Studio.
## Features
* Debug Linux Kernel with Visual Studio
* Support for Raspberry Pi 3, i.MX6 and other devices
* IntelliSense support even for GNU extensions
